基本介绍

因国内的主流在线音乐平台有个人隐私之虞，也未找到相关代码实现方便魔改且功能精简的开源项目，故自己写了一个基于浏览器的本地开源音乐播放器，主界面风格类似百度音乐盒。

使用须知

1、本应用的访问路径为index.html
2、此应用基于浏览器缓存（localStorage）工作，每次清理浏览器缓存前请及时导出配置信息；
3、隐身模式下的操作记录将不会保存，如果需要请在关闭窗口前导出配置信息；
4、每次导入配置信息将会完全覆盖当前的所有配置，如果需要请先导出当前备份；
5、播放列表新增远程媒体时请填写完整访问路径；
6、播放列表新增本地文件或者文件夹时时请先手动填写本地磁盘的访问路径的前缀，例如“file:///D:/tmp/”(windows)或者“file:///Applications/tmp/”(Mac)，然后回车键即可选择该路径下的具体文件或者文件夹；
7、上一步介绍的访问路径前缀如果不知道的，可以将本地的文件或者文件夹拖入浏览器页面中，然后复制地址栏地址即可得到该文件或文件夹的完全的访问路径；
8、媒体的添加与播放基于本地计算机与当前使用浏览器的支持，被支持的视频也可参与播放，不支持的媒体将会被自动忽略；
9、非真实访问路径的媒体缩略图片保存在浏览器缓存中，无法随配置文件导出，且应用缓存超过浏览器的缓存上限后，会删除最久未访问的缩略图缓存；
10、有任何建议或者意见可以到 github 上提issue，但我不一定有时间改:) ；
11、请勿将此项目用于商业用途，具体请参考：许可证。

功能点

1、点击缩略图可替换当前媒体的缩略图；
2、播放时缩略图将会自动旋转；
3、支持静音/取消静音；
4、支持调整音量大小；
5、支持切换上一个、下一个媒体；
6、支持暂停/播放媒体；
7、支持0.5倍/1倍/2倍的播放速度切换；
8、支持正序播放/倒序播放/随即切换/加权随机四种播放顺序的切换；
9、播放时间数字与播放进度条长度随播放的时间的推移而增长；
10、支持拖动整个播放器以调整位置；
11、支持导入本地的个性化配置；
12、支持导出当前的个性化配置；
13、支持切换主题颜色功能；
14、支持显示/隐藏播放列表界面；
15、支持展示/隐藏整个播放器；
16、支持搜索/播放/新建/重命名/清理/全部添加/删除自定义的播放列表；
17、播放列表数量过多时支持惯性拖动展示切换；
18、支持双击播放列表中的媒体来切换当前播放的媒体；
19、支持点赞/重命名/删除媒体；
20、点赞的媒体在“加权随机”的顺序时播放的概率将会加倍；
21、支持拖动媒体来调整媒体在列表中的顺序；
22、支持播放列表添加新的远程媒体/本地媒体/本地文件夹。
23、自动解析所添加本地媒体/本地文件夹物理文件中的缩略图。
24、支持拖动播放列表底部的sideBar来调整当前播放列表的展示高度。

兼容性

支持主流的Chrome、Firefox、Safari及相关内核(包括新版Edge)的浏览器，IE浏览器全系列不支持（IE 11可以查看基本的demo样式）。

二次开发

此项目的数据存储接口定义详见store.js文件中的StoreHandler对象，
默认提供了基于浏览器缓存的本地数据存储实现（local.js）。
如果想要将个人数据存储于第三方的网络，可以参考remote.js,自己实现对应的远程存储器并注册即可。
注意，该参考文件（remote.js）仅作为示例，并未引入，删除不影响使用。
